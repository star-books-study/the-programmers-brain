### 4.1.1. 작업 기억 공간과 STM 의 차이
- STM 의 역할이 **정보를 기억하는 것**인 반면, 작업 기억 공간의 역할은 **정보를 처리하는 것**이다
- 작업 기억 공간도 **특정한 문제에 적용된 STM** 이기 때문에 STM 과 같은 제한이 있다
  - 2~6개로 작업 기억 공간의 맥락에서 이 용량을 `인지 부하` 라고 부른다.
