## 11.1. 프로그래밍 중 이루어지는 다양한 활동
- CDN은 프로그래밍 언어 또는 코드베이스의 인지적 영향을 평가하고, 검색/이해/전사/증가/탐구 라는 5가지 활동을 기술한다

### 11.1.1. 검색
- 검색 : 코드베이스를 살펴보고 특정 정보를 검색하는 작업
- 검색은 STM에 무리를 가한다
  - 검색 대상, 이미 탐색한 코드의 경로 및 이유, 심층적으로 탐색할 사항을 기억해야 한다
